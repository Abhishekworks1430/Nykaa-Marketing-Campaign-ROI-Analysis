# 💄 From Impressions to Profit: A Marketing Analytics Case Study

## Overview

Marketing campaigns generate millions of impressions, clicks, and conversions, but not every campaign contributes equally to business growth.

This project analyzes more than **55,000 marketing campaigns** to understand which campaigns deliver the highest returns, which customer segments generate the most value, and where marketing budgets can be optimized.

Using **MySQL for data cleaning and KPI engineering** and **Excel for dashboard creation and exploratory analysis**, this project explores campaign profitability, conversion funnels, channel effectiveness, and customer behavior to uncover insights that support better marketing decisions.

---

## Objectives

The analysis was conducted to answer several business questions:

- Which campaign types generate the highest revenue and profit?
- Which customer segments provide the best return on investment?
- How does campaign performance change over time?
- Which channels and languages perform best?
- How efficient is the marketing conversion funnel?
- Which campaigns consume significant budget while producing weak returns?
- Do engagement levels and campaign duration influence ROI?
- Which campaigns should be scaled, monitored, reviewed, or redesigned?
